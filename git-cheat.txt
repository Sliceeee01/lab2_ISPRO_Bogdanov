То что я использовал по ходу дела в лабораторной работе -
это такие команды как
-git branch - Показывает все ветки в проекте
-git status - Показывает, какие файлы изменены
-git config --global user.name  - Устанавливает твоё имя
-git config --global user.email - Устанавливает твою почту 
-cd  - Переход в другую папку
-mkdir ........ - Создание новой папки
-git init - Превращает текущую папку в Git-репозиторий
-git add . - Готовит ВСЕ изменённые файлы к сохранению
-git commit -m "" - Сохраняет изменения с комментарием.
-git push - Отправляет сохранения
-git log - Показывает историю всех сохранений
-git log --oneline - Показывает историю сохранений в одну строку
-git switch - Переключает на другую ветку
-git branch -d ...  - Удаляет ветку с
